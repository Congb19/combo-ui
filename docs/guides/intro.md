- 简单易用
- TS 支持

### 起因 / Thinking

从初学前端算起，已经写了好多好多年了，包括学校项目和工作项目，几乎都沉浸在业务页面，用着别人的造好的各种轮子，对自己没有什么提升，简历上都摆不出什么值得一谈的项目内容来。于是自从 20 年 Vue3 出现开始，萌生了这个 Combo UI 的 idea。

这么多年，脑子里真的出现过很多很多 idea，但是懒癌时不时发作，导致它们一直在脑子里，最多最多只是开了个头。最后很多都死掉了。

希望 Combo UI 不要死掉！
